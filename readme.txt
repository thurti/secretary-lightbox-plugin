Lightbox Plugin 
by Thomas Hurtig
version: 1.1

based on The Secretary Pop Dislpayer Plugin
uses Slimbox - (c) Christophe Beyls 2007-2010; http://code.google.com/p/slimbox/




Images in one group are are displayed as a image set. For thumbnail size, captions and title the plugin uses the settings from the "settings - project" page.


Installation: Copy the the complete folder into your system/plugins folder and choose "lightbox" in the displayer dropdown menu.
